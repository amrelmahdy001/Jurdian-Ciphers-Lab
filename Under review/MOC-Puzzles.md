## CTF Resources

### Common CTF Cipher Categories
1. **Classical substitution** — Caesar, Vigenère, Affine, Atbash
2. **Classical transposition** — Rail Fence, Columnar, Scytale
3. **Symbol recognition** — Pigpen, Dancing Men, Braille, Morse
4. **Encoding** — Base64, Base32, hex, binary, URL encoding
5. **Modern crypto** — RSA with small keys, weak AES modes, hash length extension
6. **Steganography** — LSB images, spectrogram audio, whitespace text

### CTF Workflow
1. Identify the cipher/encoding → [[Cipher Identifier Quick Reference]]
2. Collect tools → [[MOC-Tools]]
3. Analyze statistically → [[MOC-Statistical-Analysis]]
4. Apply attack → [[MOC-Cryptanalysis]]
5. Validate result → does the output make sense?
