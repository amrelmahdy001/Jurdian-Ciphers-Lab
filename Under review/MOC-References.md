## External Resources

### Books
- *The Code Book* — Simon Singh; accessible history of cryptography
- *Applied Cryptography* — Bruce Schneier; practical modern cryptography reference
- *The Codebreakers* — David Kahn; comprehensive historical cryptography
- *Introduction to Modern Cryptography* — Katz & Lindell; formal/academic treatment
- *Cryptanalysis* — Helen Fouché Gaines; classical cipher breaking reference

### Online Tools (External)
- [dCode.fr](https://www.dcode.fr/en) — Comprehensive online cipher tool collection
- [Boxentriq.com](https://www.boxentriq.com) — Cipher tools and puzzle resources
- [CyberChef](https://gchq.github.io/CyberChef/) — GCHQ's Swiss Army knife for data transformation
- [Cryptii](https://cryptii.com) — Visual encoding/decoding pipeline tool
- [Rumkin.com](http://rumkin.com/tools/cipher/) — Classical cipher tools

- [CrypTool](https://www.cryptool.org) — Educational cryptography software
