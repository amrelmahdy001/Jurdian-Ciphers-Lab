# Writing Systems — Index

> **Definition:** Writing systems are alphabets, scripts, or symbolic notations for recording language. They are NOT ciphers. A writing system used to encode Latin text is a substitution cipher — see [[MOC-Symbolic-Ciphers]].
> ← Back to [[MOC-Master]]

---

## Critical Distinction

| Type | Example | Nature |
|---|---|---|
| Historical writing system | [[Elder Futhark]] | Actual alphabet used for Norse language |
| Constructed/artistic script | [[Klingon Language]] | Invented for a fictional universe |
| Symbolic cipher | [[Pigpen Cipher]] | Latin letters replaced with symbols; IS a cipher |

- [[Elder Futhark]] is a writing system — it has its own phonetic values and was used to write Norse
- [[Wingdings Font]] is a symbol font — letters replaced with symbols; used as simple substitution cipher
- [[Pigpen Cipher]] is a cipher — specifically a monoalphabetic substitution cipher using geometric symbols

## Sub-Categories

### Historical Writing Systems
> Real alphabets and scripts from human history.

- [[American Sign Language]] — Visual-gestural language using hand shapes
- [[Amharic (Ge'ez) Script]] — Ethiopian/Eritrean syllabary
- [[Braille Alphabet]] — Tactile writing system using raised dots; Louis Braille, 1824
- [[Celestial Alphabet]] — Occult alphabet derived from Hebrew; Heinrich Cornelius Agrippa
- [[Elder Futhark]] — Oldest runic alphabet; used ~150–800 CE; Norse/Germanic
- [[Enochian Alphabet]] — Angelic language and script; John Dee and Edward Kelley, 1580s
- [[Greek Alphabet]] — Foundation of Latin and Cyrillic; still used
- [[Hieroglyphs]] — Ancient Egyptian logographic script; deciphered by Champollion 1822
- [[Lingua Ignota]] — Constructed sacred language; Hildegard von Bingen, 12th century
- [[Malachim Alphabet]] — Occult alphabet; said to be the language of angels
- [[Moon Alphabet]] — Tactile alphabet for blind readers; William Moon, 1845
- [[Nyctography]] — Writing system for use in the dark; Lewis Carroll, 1891
- [[Ogham Alphabet]] — Early medieval Irish script; notches on stone edges
- [[Passing the River Alphabet]] — Occult script attributed to Agrippa
- [[Sanskrit]] — Classical Indo-Aryan language; Devanagari script
- [[Theban Alphabet]] — Witchcraft alphabet; also called the "Witch's Alphabet"
- [[Tifinagh Alphabet]] — Berber script; North Africa
- [[Younger Futhark]] — Later runic alphabet; Viking Age

- [[Daedric Alphabet]] — Elder Scrolls / Oblivion / Skyrim

- [[Futurama Alien Alphabet 2]] — Second alien alphabet in Futurama (more complex)
- [[Gnommish Language]] — Artemis Fowl book series; lower page cipher
- [[Gravity Falls Bill Cipher]] — Gravity Falls animated series
- [[Gravity Falls Journal 3 Cipher]] — Various ciphers from Gravity Falls
- [[Hexahue Alphabet]] — Six-color symbolic alphabet; puzzle/ARG use
- [[Hylian Language]] — Legend of Zelda series; multiple variants per game
- [[Illageralt Alphabet]] — Minecraft; written on banners and structures
- [[Klingon Language]] — Star Trek; constructed by Marc Okrand
- [[Kryptonian Alphabet]] — DC Comics Superman universe
- [[Matoran Language]] — Bionicle / LEGO; constructed for the toyline
- [[Sheikah Language]] — Legend of Zelda: Breath of the Wild

- [[Vulcan Language]] — Star Trek; multiple constructed scripts
- [[Wingdings Font]] — Microsoft symbol font; often used as simple substitution
