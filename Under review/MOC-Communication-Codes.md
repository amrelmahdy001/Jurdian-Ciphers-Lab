# Communication Codes — Index

> **Definition:** Communication codes are standardized systems for transmitting information across a medium — telegraph, radio, telephone, visual signals. They are NOT ciphers (they provide no secrecy) unless a separate encryption layer is added.
> ← Back to [[MOC-Master]]

---

## Sub-Categories

### Telegraphy & Early Digital
- [[Alternate Mark Inversion Code]] — Line code used in digital telecommunications
- [[Baudot Code]] — 5-bit character encoding; used in early teleprinters; Émile Baudot, 1870
- [[Decabit Code]] — 10-bit coding scheme
- [[Manchester Code]] — Self-clocking line code; used in Ethernet
- [[Morse Code]] — Dot-dash encoding of the alphabet; Samuel Morse & Alfred Vail, 1837
- [[Wabun Code]] — Japanese Morse code adaptation; dots and dashes for kana

### Telephone & Mobile
- [[DTMF Code]] — Dual-Tone Multi-Frequency; telephone keypad tones
- [[Multi-tap Phone (SMS)]] — Pre-smartphone text input; multiple presses per key
- [[Phone Keypad Cipher]] — Letters encoded by phone key number + press count
- [[T9 Predictive Text]] — Numeric key → word prediction; mobile phones

### Visual & Optical Signaling
- [[Chappe Code]] — French optical telegraph; arm positions on towers; Claude Chappe, 1792
- [[Flag Semaphore]] — Letter encoding using flag positions; arm angles = letters
- [[International Maritime Signals]] — Flag codes for ship-to-ship communication
- [[Navy Signals Code]] — Military naval signal codes
- [[Semaphore Angle Code]] — Numeric angle system for semaphore

### Phonetic & Voice
- [[NATO Phonetic Alphabet]] — Alpha, Bravo, Charlie…; standardized spoken letter identification
- [[PGP Word List]] — Word pairs for verifying cryptographic fingerprints verbally

### Other Codes
- [[Parsons Code]] — Describes melody by note direction (U/D/R); music search code
- [[Tom Tom Code]] — Drum-based communication code
